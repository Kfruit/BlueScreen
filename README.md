# Bluescreen
只是一个假装电脑蓝屏的小程序，并且支持重启进度、终止代码和二维码链接的修改
### 如何使用
在启动程序后，修改自定义信息，然后按下右下方的开始按钮
### 如何退出
在蓝屏界面下，按下Ctrl+Q退出界面
